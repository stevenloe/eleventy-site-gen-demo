## General info
This project is a demo of static site generation with Eleventy
	
## Technologies
Project is created with:
* Eleventy version: 0.11.0
	
## Setup
To run this project, install it locally using npm:

```
$ cd ../eleventy-site-gen-demo
$ npm install
$ npm start
```
